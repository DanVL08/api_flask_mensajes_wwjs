PASOS:
1.- Clone el repositorio
2.- ACTIVA LOS PERMISOS PARA USAR SCRIPTS: Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope Process
3.- CREE  ENTOTNO VIRTUAL: .\env\Scripts\activate 
4.- INSTALE LOS REQUERIMIENTOS pip install -r requirements.txt
5.- ACTIVA LOS SERVIDORES DE XAMPP: Apache / MySQL
6.- INCIA LA APLICACION  DESDE LA CARPETA src/: python app.py
